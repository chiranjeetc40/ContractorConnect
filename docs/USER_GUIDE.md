## ğŸ“± Using the App

### For Society Users

#### Step 1: Create a Request
1. Tap the â• **"New Request"** button (bottom-right)
2. Fill in details:
   - **Title**: e.g., "Fix Leaking Pipe in Bathroom"
   - **Category**: Select (Plumbing, Electrical, etc.)
   - **Description**: Detailed explanation (min 50 characters)
   - **Location**: Address, City, State, Pincode
   - **Budget**: Min and Max range
3. Tap **"Submit"**

#### Step 2: Review Bids
1. Go to your request
2. See list of bids from contractors
3. View contractor details
4. Compare prices and proposals

#### Step 3: Accept a Bid
1. Choose the best bid
2. Tap **"Accept Bid"**
3. Contractor is assigned automatically
4. Work status changes to "In Progress"

### For Contractors

#### Step 1: Browse Work
1. Open app â†’ See all available requests
2. Use search bar to find specific work
3. Filter by category or location
4. Tap any request to see details

#### Step 2: Submit a Bid
1. View request details
2. Tap **"Submit Bid"**
3. Enter:
   - Your bid amount
   - Estimated timeline
   - Work plan/notes
4. Tap **"Submit"**

#### Step 3: Track Bids
1. Go to **"My Bids"** tab
2. See all your submitted bids
3. Check status: Pending | Accepted | Rejected
4. View assigned work
